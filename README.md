## Fonctionnalités

- Notes + moyenne (par matière + moyenne gé)
- Devoirs
- Espaces de travail
- Carnet de correspondance
- Vie scolaire (absences, retards, colle ...)

## Utilisation
Executer le raccourci dans la feuille de partage du site ecoledirecte dans safari.
